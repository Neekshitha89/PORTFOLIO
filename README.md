# 🌐 Neekshitha Portfolio Website

## 📌 Overview

This is a personal portfolio website created using HTML and CSS. It showcases my profile, skills, projects, achievements, and contact information in a clean and modern design.

---

## 🎯 Features

* ✨ Modern UI with gradient background
* 🧊 Glassmorphism card design
* 📱 Responsive layout
* 🧑‍💻 Home section with image and introduction
* 📊 Skills with progress bars
* 💼 Projects section with links
* 🏆 Achievements section
* 📞 Contact section with clickable links
* 🎨 Neon glow effects and hover animations

## 🛠️ Technologies Used
